LogicAppAnalytics

Outil d'analyse pour data analysts

← Retour à l'accueil

Contexte

Les équipes d'analystes de données au sein du Service des Laboratoires avaient besoin d'un outil pour analyser rapidement les données sans passer par des processus lourds et répétitifs. Les solutions existantes étaient soit trop complexes, soit trop limitées.

Solution : LogicAppAnalytics

J'ai développé LogicAppAnalytics, une application conçue pour :

  • Analyser rapidement des données de laboratoire
  • Traiter à la fois des données statistiques et géospatiales
  • Générer des visualisations automatiques
  • Exporter les résultats dans différents formats

Fonctionnalités principales

  • Import de données : Support de multiples formats (CSV, Excel, Shapefiles)
  • Analyse descriptive : Statistiques automatiques, détection d'anomalies
  • Analyse géospatiale : Cartographie automatique, calculs de distances
  • Visualisation : Graphiques interactifs, cartes, tableaux de bord
  • Export : Rapports PDF, images, données nettoyées

Architecture technique

  • Backend : Python (Flask) avec Pandas et GeoPandas
  • Frontend : HTML, CSS, JavaScript avec Plotly pour les graphiques
  • Base de données : SQLite pour les sessions utilisateur
  • Déploiement : Conteneur Docker pour faciliter l'installation

Technologies utilisées

Python R Flask Pandas GeoPandas Plotly Docker

Impact

  • Réduction du temps d'analyse de 70% pour les traitements courants
  • Standardisation des méthodes d'analyse entre les analystes
  • Adoption par 15 data analysts du MINSANTE et d'autres projets
  • Reproductibilité : Les analyses peuvent être partagées et reproduites facilement

Témoignage

"LogicAppAnalytics nous a fait gagner un temps considérable. Ce qui nous prenait une journée entière se fait maintenant en 2 heures." - Data Analyst, MINSANTE
